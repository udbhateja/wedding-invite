# Sophia & Robert Wedding Invite Clone

This project replicates the Crafto "Sophia & Robert" wedding invitation landing page.

## Getting Started

1. Install any tooling you prefer for static development (optional).
2. Open `index.html` directly in a browser, or run a simple static server such as `npx serve` from the project root.
3. Edit assets inside `assets/`, styles in `styles/main.css`, and scripts in `scripts/main.js`.
